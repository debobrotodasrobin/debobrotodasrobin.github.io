
# Guide for Personal Website

## Adding New Projects

1. Create a new directory inside `/projects`.
2. Add an `index.md` file with details about the project.
3. Update `/projects/index.html` to include a link to the new project.

## Adding New Blogs

1. Create a new directory inside `/blog`.
2. Add an `index.md` file with details of the blog post.
3. Update `/blog/index.html` to include a link to the new blog post.

## Adding CV Information

Update the content in `/cv/index.md` as needed.

### Example Project Structure:

```bash
/
├── cv/
│   └── index.md
├── projects/
│   ├── project_1/
|      └── index.md
│   └── index.html
├── blog/
│   ├── post_1/
│   │   └── index.md
│   └── index.html
└── _config.yml
```
